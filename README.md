# Блог (Frontend)

Демо:  https://chat-lyart-gamma.vercel.app/

## Какие технологии используются:
- React.js (Хуки)
- Redux
- Redux-thunk
- React-hook-form/Yup
- Axios
- Moment
- Material UI/Emotion 

## Что реализовано:
- Авторизация, Регистрация (JWT)
- Добавление друзей (Invite)
- Список друзей
- Валидация форм (схема валидации через Yup)
- Индикаторы онлайн и не в сети
